
# Cape assessment

This app is made for Cape. You can see animations and adjust the header, color and image size. 


## Installation

Download or clone the files and install/run the app with npm

```bash
  npm install 
  npm run start
```
    
## Deployment

To deploy this project run

```bash
  npm run build
```
